# HTH Route AI

HTH Route AI is an independent AI gateway for developers who need low-cost OpenAI-compatible coding model access.

It provides:

- OpenAI-compatible API access
- Usage metering
- Spend limits
- Request logs
- Routing for coding-agent workflows
- Integration guides for Cline, OpenCode, Continue, and Aider
